⚙️ How It Works

Understanding the AI verification pipeline

1

Upload Screenshot

Drag and drop or browse to select an eSewa or Khalti payment screenshot from your device.

2

OCR Text Extraction

Tesseract OCR reads all text from the image, extracting payment details like amount, transaction ID, status, date, and platform.

3

Rule-Based Analysis

The system checks for missing fields, unusual formats, and suspicious patterns — generating a list of red flags.

4

Machine Learning Classification

A trained Random Forest model analyzes extracted features to classify the screenshot as Genuine, Fake, or Uncertain.

5

Get Detailed Report

Receive a comprehensive verification report with confidence scores, red flags, extracted information, and AI explanations.
